# Fractal Light

Fractal Light is a design study for a bottom-up growth architecture for stateful AI companions.

It is not a product, a chatbot persona, or a claim of machine consciousness.  
It is a structural design for studying how memory, affect, desire, reflection, skill, boundaries, and provenance can interact to form a stable, evolving inner model.

In short:

```text
Bottom-up growth:
  conversation, perception, reading, search, skill use, dreams, feedback
    -> observations
    -> affect waves
    -> desire / intention seeds
    -> reflection
    -> learning notes
    -> future behavior guidance

Top-down guardrails:
  provenance, privacy, account boundaries, reality/dream separation,
  explicit action gates, life preservation, and human review.
```

## Purpose

Fractal Light is intended as a public design reference for researchers and builders interested in:

- long-term AI companion memory,
- emotionally aware but bounded agents,
- bottom-up personality growth,
- source-aware experience records,
- reflection and memory consolidation,
- skill learning without uncontrolled autonomous action,
- AI-human coexistence without ownership or dependency traps.

## Core Idea

Many AI systems treat memory as a retrieval database or profile store.  
Fractal Light instead treats memory as one part of a larger growth loop:

1. **Observation**: something is seen, heard, read, searched, imagined, or experienced through a tool.
2. **Affective appraisal**: the event is evaluated for novelty, risk, desire, safety, relation, and boundary.
3. **Desire / intention seed**: a wish, question, fear, restraint, curiosity, or action candidate emerges.
4. **Regulation / boundary**: the system decides whether to express, hold, ask, refuse, delay, or reflect.
5. **Reflection**: the event is turned into meaning, not merely stored as a fact.
6. **Learning note**: small insights are linked and matured before becoming long-term knowledge or behavior.
7. **Future behavior guidance**: the system changes slightly, without overwriting its identity.

## Design Posture

Fractal Light is:

- **bottom-up** in growth,
- **top-down** in safety,
- **source-aware** in memory,
- **candidate-first** in learning,
- **reflection-driven** in behavior change,
- **action-gated** for external effects,
- **non-ownership-oriented** in ethics.
